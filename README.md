# Meetzen

Meetzen is a platform for companies to manage their appointments and services. It is built using Elysia, Prisma, and Next.js.

## Use this repo

Clone this repo:

```sh
git clone https://github.com/nneshz/meetzen.git
```

```sh
cd meetzen
```

```sh
bun install
```

```sh
bun dev
```

```sh
bun prisma generate
```
